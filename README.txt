How to run the code:


You need to have an API key and put the API key into OPENAI_API_KEY and FLASK_SECRET_KEY values in the apikey.py file (it costs money to have an API key, so if you need me to run it on my own API key just send me what you want me to run and I can run it myself, or you should be able to use the professors API key if you are on campus)

You then need to have flask and python 3.13 installed on your pc

then you need to go into windows PowerShell and run the app.py file from there (i usually have to cd into the actual file and then select the python 3.13 version to run the application, although that's because i have multiple versions of python installed)

after you have the local server running you need to go into a web browser and go to the web address http://localhost:5000 and then you should be able to access the program
